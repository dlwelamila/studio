# **App Name**: tasKey: On-Demand Assistance

## Core Features:

- Task Posting: Customers can post tasks with category, description, location, and budget range.
- Helper Profile Verification: Administrators verify helper profiles to ensure safety and trust on the platform.
- Task Offer Submission: Helpers submit structured offers (price, availability, message) for available tasks.
- Offer Review and Selection: Customers review offers from helpers and select the most suitable one.
- Task Status Tracking: Track task progress from assignment to completion with status updates.
- Feedback and Rating: Customers provide feedback and ratings upon task completion to ensure quality and accountability.
- Task Recommendation: AI-powered tool suggests helpers that fit the requirements for a certain posted task. Considerations are made to the area, user ratings and estimated time of delivery.

## Style Guidelines:

- Primary color: Deep sky blue (#43b5ff), evoking a sense of reliability and trustworthiness in the platform.
- Background color: Light cyan (#e5f5ff), providing a clean and calm backdrop to the user interface.
- Accent color: Midnight blue (#094067), used sparingly to highlight key interactive elements and calls to action, adding sophistication to the design.
- Body font: 'PT Sans', a humanist sans-serif, chosen for readability and modern appeal, ideal for descriptions and general text throughout the platform.
- Headline font: 'Space Grotesk', a proportional sans-serif, lending a tech-oriented feel to headlines.
- Simple, clear icons that represent task categories and actions. Consistent in style, using line icons with rounded edges for a friendly feel.
- Clean and intuitive layout with a focus on easy navigation. Cards for task listings and offers to provide clear visual separation.
- Subtle animations for task status updates and loading indicators to enhance user experience without being distracting.
